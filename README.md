# Photacles

Private Photo Cloud for Home Server.

## Install

### Before start

Install:

+ [nodeJS][1]
+ [yarn][4]
+ Console image editor: [graphicsmagick][2]
+ Database: [postgresql][3]



[1]: https://nodejs.org
[2]: http://www.graphicsmagick.org
[3]: https://www.postgresql.org
[4]: https://yarnpkg.com
